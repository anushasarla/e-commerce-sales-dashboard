# E-Commerce Sales Dashboard

## Overview
This Power BI dashboard provides an interactive view of e-commerce performance by tracking sales, profit, quantity, customers, states, payment modes, categories, and monthly trends.

## Key Highlights
- Sales Amount: 438K
- Profit: 37K
- Quantity Sold: 5615
- AOV: 121K

## Features
- Monthly profit analysis
- State-wise sales comparison
- Customer-wise sales tracking
- Category and sub-category insights
- Payment mode distribution
- Quarter-based filtering

## Tools Used
- Power BI Desktop

## Purpose
The dashboard helps users quickly understand business performance and support data-driven decision-making.
